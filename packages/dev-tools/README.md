# @mvila/dev-tools

Simple development tools used by mvila's projects.

## Installation

```bash
npm install @mvila/dev-tools --save-dev
```

## Usage

To build a TypeScript project, add the following in your `package.json` file:

```json
{
  "scripts": {
    "build": "dev-tools build:ts-library"
  }
}
```
